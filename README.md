# RandomPicGenerator
RandomPicGenerator to generate random pic for DP on social media or wallpaper for your phone.

## Usage example: 
    Run the code python colouring_board.py
    Input a number for resolution in the GUI box
    The image is automatically saved as random_art.jpg
   
## Starting off
Install the dependencies for the project
pip install --user -r requirements.txt 

####Enjoy!
