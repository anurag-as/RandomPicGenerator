# RandomPicGenerator
RandomPicGenerator to generate random pic for DP on social media or wallpaper for your phone.

![Screenshot](https://raw.githubusercontent.com/sampathanurag/RandomPicGenerator/master/Project%20Screenshots/ProjectGeneratedImages.png)

## Usage example: 
Run the code
```
python colouring_board.py
```
Input the resolution and the number of images in the text box.
The image is automatically saved as **random_art_\*.jpg**
    
## Starting off
Install the dependencies for the project<br>
```
pip install --user -r requirements.txt
```
Enjoy!
